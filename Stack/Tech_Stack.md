# Стек Технологий

- **Язык**: Python 3.10+
- **Окружение**: Kaggle Environments (`kaggle-environments`)
- **Библиотеки**:
    - `numpy` для быстрых математических операций и геометрии.
    - `math` для базовой геометрии.
    - `scipy` (опционально) для пространственных расчетов, если потребуется.
