
# Orbit Wars

**Orbit Wars** — это многопользовательская (или синглплеерная) игра на базе физики орбитального движения. Игроки используют гравитацию небесных тел для маневрирования и уничтожения противников.

**Статус проекта:** Инициализация (Миграция на агентскую схему)
**Стек:** Python, Pygame, Orbital Physics

## Архитектура базы знаний (Obsidian)

Проект поддерживается через древовидную структуру знаний:

*   **[`/Context`](./Context)** — Игровая механика, расчеты гравитации и ТЗ.
*   **[`/Stack`](./Stack)** — Спецификации Pygame, константы физики и дизайн API.
*   **[`/Structure`](./Structure)** — Карта классов (Planet, Ship, Projectile) и их взаимодействие.
*   **[`/Data`](./Data) [LOCAL]** — Ассеты, логи тестов и сырые идеи.
*   **[`/Private`](./Private) [LOCAL]** — Заметки об ошибках сборки (напр. CMake/библиотеки).

## Технический стек и физика

В основе игры лежит закон всемирного тяготения Ньютона, используемый для расчета траекторий:

$$F = G \frac{m_1 m_2}{r^2}$$

Где:
*   $G$ — гравитационная постоянная;
*   $m_1, m_2$ — массы тел;
*   $r$ — расстояние между центрами масс.

Для реализации используется библиотека **Pygame**. Все расчеты векторов производятся в реальном времени.
